# Investor / Partner Deck Outline (v1)

## 1) Title & Positioning
- Human-guided multi-agent studio for video, music, and AI apps.
- Proprietary orchestrator, policy engine, and template/IP stack.

## 2) Problem
- Brands need bespoke AI content/apps without black-box risk.
- Current tools lack policy, consent, watermark, and auditability.

## 3) Solution
- Company OS: intake → orchestrator → creative/tech agents → QA/policy → delivery.
- Built-in safety (consent, watermark, budget/rate guards) and file-scoped ops.

## 4) Product
- Control dashboard: Projects, Agents, Templates/Prompts, Music/Lyrics, Memory, Policy/Safety, Analytics, Deployment, Files, Company OS.
- Agent actions: create tasks/events/notes/files; assemble video/music; enforce policy.
- File workspace scoped to /workspace; audited actions.

## 5) Tech Moat
- Orchestrator + policy engine + prompt/template system.
- Video pipelines (scene prompts, model selection, EDL/assembly, watermark).
- Songwriting pipelines (structure JSON, lyric drafts, vocal analysis).
- Client memory and IP store.

## 6) Traction / Proof
- (Add) Demos: video/music outputs, app agents, policy logs.
- (Add) Early adopters/pilots and KPIs (cost/time saved).

## 7) Go-To-Market
- Direct to brands/creators; agency partnerships.
- Website AI agent for intake and upsell.

## 8) Business Model
- Platform + usage; optional pro services for custom agents/pipelines.
- Compliance/policy add-ons.

## 9) Roadmap
- Agent SOPs → Policy hardening → Cost/analytics → Deploy pipelines → SaaS packaging.
- Multi-model support (Sora/Hailuo/Pika) and eval loops.

## 10) Team
- (Add) Founders, domain experts, advisors.

## 11) Ask
- Funding/partnership amount; use of funds (product, safety, GTM).
